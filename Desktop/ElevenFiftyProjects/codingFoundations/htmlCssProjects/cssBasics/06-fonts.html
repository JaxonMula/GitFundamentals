Font

There are 2 font family names:

  • Generic Family font families that have the same look (ex: Serif)
  • Font Familyspecific font family (ex: Times New Roman)

Font family will have several fonts that it can fall back on if there is no browser support

Names must be held w/n "" like "Times New Roman"
