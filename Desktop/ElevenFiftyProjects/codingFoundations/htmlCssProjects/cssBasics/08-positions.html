Position

The position property specifies the type of positioningmethod used for an element.

4 states of positioning that an element can be in

  • Fixed This pins an element to a specific spot. doesnt move regardless of scrolling
  • Statisc This is the default setting
  • Relative Doesn't take the element out of "document flow" and still apears in its space/dic.container, but allows it to be offset a bit from the top, bottom, right, and left
  • Absolute Breaks document flow. if the parent element is set to relative and the child is absolute then the child will be at the coordinates (0,0)

The z-index gicen the illusion that one elemnt is on top of the other. this value can be either positive or negative. the lesser value is on the bottom.
