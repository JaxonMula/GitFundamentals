Width and Height

Width and height properties are used to set the width and height of an element.

The default setting for width and height is auto, browser decides the width and height

Some other settings:

  • Specified length values such as px and em
  • Percentage of the containing element (ex:50%,100%,etc)

This element has a width of 50% and a height of 200px
